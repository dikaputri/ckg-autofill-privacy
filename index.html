📋 Pendahuluan

CKG Autofill adalah ekstensi Chrome yang dikembangkan untuk membantu petugas administrasi kesehatan dalam mengisi formulir pendaftaran program Cek Kesehatan Gratis (CKG) di situs resmi Sehat Indonesiaku (sehatindonesiaku.kemkes.go.id) secara otomatis menggunakan data dari file Excel atau CSV.

Kebijakan privasi ini menjelaskan data apa yang dikumpulkan, bagaimana data tersebut digunakan, di mana data disimpan, dan hak-hak pengguna terkait data tersebut.

Ekstensi ini hanya diperuntukkan bagi petugas resmi yang telah memiliki token aktivasi yang valid.

🗃️ Data yang Dikumpulkan

Berikut rincian data yang diproses oleh ekstensi ini:

Jenis Data Contoh Disimpan di
Identitas pasien NIK, nama lengkap, tanggal lahir, jenis kelamin Lokal (IndexedDB)
Token aktivasi Format DPM-XXXXXX-0000 Lokal (chrome.storage)
Email pengguna Untuk registrasi & cek status akses Server validasi
Device ID ekstensi ID unik instalasi Chrome Server validasi
Riwayat web / lokasi / keuangan — Tidak dikumpulkan

⚙️ Cara Penggunaan Data

  • Data identitas pasien digunakan hanya untuk mengisi otomatis field formulir pendaftaran CKG di sehatindonesiaku.kemkes.go.id.
  • Token aktivasi digunakan untuk memverifikasi bahwa pengguna adalah petugas resmi yang berwenang menggunakan ekstensi ini.
  • Email digunakan untuk proses pendaftaran akses dan pengiriman token oleh administrator.
  • Device ID digunakan untuk membatasi satu token per perangkat demi keamanan akses.
Data pasien tidak pernah dikirim ke server mana pun. Semua data pasien hanya tersimpan secara lokal di perangkat pengguna dan tidak meninggalkan browser.

🔒 Penyimpanan & Keamanan

  • Data pasien disimpan di IndexedDB browser — sepenuhnya lokal, tidak dapat diakses dari luar perangkat.
  • Token & cache validasi disimpan di chrome.storage.local — hanya dapat dibaca oleh ekstensi ini.
  • Validasi token dilakukan melalui koneksi HTTPS terenkripsi ke server resmi.
  • Tidak ada database cloud, tidak ada analitik pihak ketiga, tidak ada iklan.
Ekstensi tidak menggunakan layanan pihak ketiga seperti Google Analytics, Firebase, Sentry, atau sejenisnya.

🚫 Data Tidak Dibagikan ke Pihak Ketiga

Kami menyatakan dengan tegas bahwa:

  • Data pengguna tidak dijual kepada pihak mana pun.
  • Data pengguna tidak ditransfer ke pihak ketiga di luar keperluan validasi token yang telah dijelaskan.
  • Data pengguna tidak digunakan untuk tujuan iklan, profiling, atau analitik komersial.
  • Data pengguna tidak digunakan untuk menentukan kelayakan kredit atau pinjaman.

👤 Hak Pengguna

  • Hapus data kapan saja — pengguna dapat menghapus data pasien melalui tombol "Hapus Data" di dalam panel ekstensi.
  • Cabut akses — token dapat dinonaktifkan oleh administrator atas permintaan pengguna.
  • Uninstall — menghapus ekstensi akan menghapus semua data lokal secara otomatis.
